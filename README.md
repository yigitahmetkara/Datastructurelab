Data Structures Lab Preparation (Veri Yapıları Laboratuvar Hazırlığı)
Bu depo, Veri Yapıları (Data Structures) dersi laboratuvar çalışmaları için hazırladığım Java uygulama örneklerini ve temel algoritma çözümlerini içermektedir. Proje, temel veri yapılarını anlamak ve laboratuvar sınavlarına/ödevlerine pratik yapmak amacıyla geliştirilmiştir.

İçerik ve Uygulamalar
Bağlı Listeler (Linked Lists): Tek yönlü ve çift yönlü bağlı liste operasyonları.

Yığın ve Kuyruk (Stack & Queue): LIFO ve FIFO prensiplerine dayalı veri yönetimi.

Ağaç Yapıları (Trees): İkili arama ağaçları (BST) ve temel ağaç gezinti (traversal) algoritmaları.

Sıralama Algoritmaları: Farklı karmaşıklıktaki sıralama yöntemlerinin Java uygulamaları.

Teknik Detaylar
Geliştirme Ortamı: Eclipse IDE

Programlama Dili: Java

Versiyon Kontrolü: Git & GitHub
